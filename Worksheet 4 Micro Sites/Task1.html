The Browser window object allows JavaScript to make changes to the client side Browser Object Model. As a result, we can make websites more interactive as it enables changes to occur client side in response to events which take place client side.

Below are a few working examples which outline the behavious which the window object can exhibit.

The first example shows how we can create a new window object using JavaScript, some parameters have been passed into the method to generate the child window.

The next example will return the port for the page.

Search is another property which we can return providing the URL contains a query string.

The syntax for search is location.search, should you wish to set the search property the syntax is, location.search=querystring.

The location object also has some associated methods, the first of which is assign(), using this we can assign a new URL to open.

The reload() method reloads the current page.

The replace() method is alike assign() however, replace() removes the URL from the document history and as a result the back button will not work.
